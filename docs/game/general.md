# ChemGuesser 化学猜盐 游戏规则

游戏开始前，设置各个题目池的权重。常用权重设置如下：

1. 入门模式：easy:0.9, mid-easy:0.1, mid:0
2. 简单模式：easy:0.5, mid-easy:0.5, mid:0
3. 进阶模式：easy:0.2, mid-easy:0.8, mid:0
4. 中等模式：easy:0.1, mid-easy:0.8, mid:0.1
5. 困难模式：easy:0, mid-easy:0.3, mid:0.7

游戏开始时，系统从所有题库（src.game.ProblemLibs，亦称作“题目池”）中按设置的权重从easy、mid-easy、mid中随机选取一个题目池，然后在其中随机选取一个化学物质作为“题目”。该物质不会被玩家提前知道。

之后，玩家可以进行有限次或无限次（由游戏模式决定）提问，每次提问一个有关该物质的问题（例如：该物质是否是氧化物？），然后调用src.ai.playerguess向AI发送提问，AI会根据“题目”物质和玩家的提问给出回答。

具体地，若玩家提问了有关该物质特征或性质的问题，若该物质具有这个特征或性质，则AI会回答“YES”，若不具有，则回答“NO”；若玩家直接猜测该物质的名称，若正确，则AI会回答“CORRECT”，若不正确，则AI会回答“INCORRECT”；若玩家的问题不符合以上两种或无法用“YES”或“NO”回答（例如：该物质的摩尔质量是多少？），则AI会回答 CANNOT ANSWER。

若玩家猜对了该物质（即AI回答“CORRECT”），则本轮胜利。胜利和失败条件由具体游戏模式决定。

如需进一步了解AI回答的机制，可在文档config/prompt.md中阅读其系统提示词。
